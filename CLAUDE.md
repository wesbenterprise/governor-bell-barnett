# Claude Code Project Instructions

## Git Workflow
- Always develop and push directly on `main`. Do not create feature branches unless explicitly asked.
- Use clear, descriptive commit messages.

## Style
- Dark, elegant, professional design. Navy/gold/white color palette (Florida/governance aesthetic, NOT neon).
- Orbitron for headings, Inter for body text.
- Left sidebar navigation (Wesley's preferred layout).
- Responsive — works on mobile.
- All time displays: relative first + absolute in parentheses.
